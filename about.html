I made this as a web app initially because all the other Random Verse Generators were either not clear about their method (how they got their "random" number, or if they were clear they didnt have open source code -- so no way for me to verify they werent just feeding me "feel good" verses.

If you want to view the source code for this project go to this link where you will find the github repository containing all the files to this website, but to make it easier for you the relevant portion is this explanation for the function I use to find a random verse:

`Math.random()` in JavaScript produces pseudo-random numbers, not truly random ones. These numbers are generated using an algorithm and are considered "pseudo" because they are deterministic - given the same initial conditions, they will produce the same sequence of numbers. However, for most practical purposes, the randomness is sufficient, especially when used for applications like generating random numbers for games, simulations, or selecting items from an array as in the case of the `fetchRandomVerse` function. For cryptographic purposes or scenarios requiring high-security randomness, other methods beyond `Math.random()` are recommended.
